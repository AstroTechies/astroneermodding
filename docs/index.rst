Astroneer Modding Documentation
===============================

This documentation website includes information and guides to help you make mods for the video game Astroneer, along with the standards that Astroneer mods should adhere to.

If you need help, feel free to check out the `Astroneer Modding Discord server <https://discord.gg/bBqdVYxu4k>`_.

Formal standards can be found in the :doc:`standards/index` section which includes the :ref:`metadatav2` and the :ref:`indexfile`.

If you want to learn how to actually make mods, check out the :doc:`guides/index` section.
